# Eltanin Core

It is a set of unix utilities, aiming to be POSIX compliant, while being
secure, small and fast.

#### Binaries
* basename
* cat
* chgrp
* chmod
* chown
* cksum
* cp
* dirname
* du
* echo
* env
* false
* head
* link
* ln
* mkdir
* mv
* pwd
* rm
* rmdir
* true
* uname
* unlink
* yes

#### Build
```
	## Separated Binaries
	$ make
	# make install
	## Single Binary
	$ make ecore
	# make install-ecore
```
